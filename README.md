# JSON Beautifier Pro

> 🎯 A powerful JSON formatter, validator, and beautifier — right in your browser.

[![Chrome Web Store](https://img.shields.io/chrome-web-store/v/placeholder)](https://chrome.google.com/webstore)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## ✨ Features

- ✅ **One-click format** — Highlight any JSON on a webpage, right-click → Format JSON
- ✅ **Real-time validation** — Invalid JSON gets red highlighting with error messages
- ✅ **Beautify & Minify** — Toggle between pretty-printed and minified views
- ✅ **Dark/Light Theme** — Follows your system preference
- ✅ **Auto-detect** — Finds JSON blocks on pages automatically
- ✅ **History** — Last 20 formatted JSONs saved locally
- ✅ **Keyboard Shortcut** — `Ctrl+Shift+J` to open
- ✅ **Works Offline** — No internet needed
- ✅ **Privacy First** — Zero data sent anywhere, everything local

## 🚀 Installation

1. Go to [Chrome Web Store](https://chrome.google.com/webstore) (coming soon)
2. Click "Add to Chrome"
3. Use `Ctrl+Shift+J` or click the extension icon

### Manual Install (Developer Mode)

1. Download `json-beautifier-pro.zip`
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder
